## 📖 نظرة عامة | Overview

موقع إلكتروني احترافي لوكالة **CyberPath Digital** المتخصصة في **التصميم الإبداعي والهوية البصرية**. تم تصميم الموقع ليعكس هوية الوكالة الإبداعية مع تقديم تجربة مستخدم سلسة وجذابة.

**Creative agency website for CyberPath Digital, specializing in Creative Design & Branding services.**

---

## ✨ المميزات | Features

### 🎨 التصميم | Design
- **ثيم داكن عصري** مع تدرجات لونية جذابة
- **Glassmorphism** تأثيرات زجاجية حديثة
- **رسوم متحركة دقيقة** لتحسين تجربة المستخدم
- **تجاوب كامل** مع جميع أحجام الشاشات

### ⚡ الأداء | Performance
- **تحميل سريع** مع تقنيات Lazy Loading
- **صور محسّنة** بصيغة WebP
- **تصغير الملفات** لتقليل حجم التحميل
- **نقاط Lighthouse** تفوق 95

### 🔍 تحسين محركات البحث | SEO
- **علامات Meta** محسّنة بالكامل
- **JSON-LD Schema** لتحسين الظهور في نتائج البحث
- **Sitemap.xml** و **robots.txt**
- **Open Graph** و **Twitter Cards** لمشاركة أفضل

### ♿ إمكانية الوصول | Accessibility
- **HTML دلالي** لتحسين قراءة الشاشات
- **تنقل عبر لوحة المفاتيح**
- **تسميات ARIA** للعناصر التفاعلية
- **توافق مع WCAG 2.1**
